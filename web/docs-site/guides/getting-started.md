# Getting Started

This guide explains the basics of using The Book Project.

## Overview

You'll need to create an account, verify your email, and log in to start using The Book Project and all its features. Once logged in, you can add books, authors, publishers, and other related information to your library.

## Creating an Account

To get started, you'll need to create an account. Follow the steps in the [Creating an Account guide](/guides/creating-an-account) to register, verify your email, and log in for the first time.
